# HOS Load Calculator PRO Plus

## Version
Master-v7 (PRO Plus Development Version)

## Product Family

This repository contains the full-feature development version of the HOS Load Calculator Progressive Web App (PWA).

The product is planned as three tiers:

- Base — Free
- PRO — $14.99
- PRO Plus — $29.99

The PRO Plus Master-v7 repository serves as the full-feature source version from which lower-tier versions can be created.

---

# Version Features

## Base Version (Free)

Core trip planning and HOS support features:

- Trip ID
- Average MPH selection
- Trip start time
- Deadhead miles
- Loaded miles
- Shipper appointment
- Consignee appointment
- Shipper stop type
- Consignee stop type
- ETA/PTA planning calculations
- Calculate function
- Reset function
- Mobile-friendly PWA design

---

## PRO Version ($14.99)

Everything included in Base, plus:

- 70-Hour / 8-Day clock tracking
- On Duty + Driving clock support
- Optional STOP 02 (mid-stop)
- Miles split tracking
- ETA/PTA support for additional stop
- Optional fuel stop

---

## PRO Plus Version ($29.99)

Everything included in PRO, plus:

- Expanded duty status tracking:

Additional planning tools:

- Split Sleeper Analysis

## Purpose
The HOS Load Calculator is a Progressive Web App (PWA) designed to help commercial drivers plan trips using Hours of Service (HOS) considerations.

This tool is intended for planning and training purposes and does not replace official ELD records, company policies, or FMCSA-required documentation.

## Current Features

- Trip planning calculations
- Deadhead and loaded miles
- Estimated arrival calculations
- HOS planning support
- Break timing considerations
- HOS clock tracking
- 70-hour weekly tracking logic
- Mobile-friendly PWA structure
