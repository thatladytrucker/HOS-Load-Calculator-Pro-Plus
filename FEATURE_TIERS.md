# HOS Load Calculator Feature Tiers

Version: Master-v7.0.0

This document defines the features included in each product edition.

---

# BASE — Free Version

## Core Trip Planning and HOS Support

Included Features:

- Trip ID
- Average MPH selection
- Trip Start Time
- Deadhead Miles
- Loaded Miles
- Total Miles Calculation
- Shipper Appointment
- Consignee Appointment
- Shipper Stop Type
- Consignee Stop Type
- ETA Planning Calculations
- PTA Planning Calculations
- Calculate Function
- Reset Function
- Mobile-Friendly PWA Design

---

# PRO — Paid Version

Includes all BASE features.

Additional Features:

## HOS Planning

- 70-Hr / 8-Day Clock
- ON DUTY tracking
- DRIVING tracking

## Load Planning

- Optional STOP 02 mid-stop
- Miles split calculation
- ETA/PTA calculation

## Fuel Management

- Optional fuel stop

---

# PRO PLUS — Premium Version

Includes all BASE and PRO features.

Additional Features:

## Expanded Duty Tracking

- On Duty
- Driving
- Break (30 min)
- Off Duty
- Sleeper Berth
- Off (End Day)

## Advanced Planning Tools

- Split Sleeper Analysis
- Recaps
- Latest Dispatch Time
- Appointment Feasibility
- Enhanced load planning
- Run-status decision support

---
