# HOS Load Calculator Product Features

## Product Family

The HOS Load Calculator is structured as a three-tier product family.

---

# Base Version — Free

Purpose:
Provide basic HOS trip planning tools.

Features:

- Trip ID
- Average MPH selection
- Trip Start Time
- Deadhead Miles
- Loaded Miles
- Total Miles Calculation
- Shipper Appointment
- Consignee Appointment
- Stop Type Selection
- ETA Calculation
- PTA Calculation
- Basic appointment timing feedback
- Recent Trips (last 8)
- Reset and Calculate functions

---

# PRO Version — $14.99

Includes all Base features.

Additional features:

- 70-Hour / 8-Day clock tracking
- On Duty tracking
- Driving tracking
- Optional STOP 02
- Miles split planning
- Expanded ETA/PTA calculations
- Fuel Stop option

---

# PRO PLUS Version — $29.99

Includes all Base and PRO features.

Additional features:

- Advanced HOS planning
- Full duty status tracking:
  - On Duty
  - Driving
  - Break
  - Off Duty
  - Sleeper Berth
  - End Day
- Split Sleeper Analysis
- Recaps
- Latest Dispatch Time Analysis
- Appointment Feasibility
- Enhanced load planning decision support

---

## Master-v7 Development Version

Master-v7 currently represents the PRO PLUS development build.

Future releases will use this foundation to create the Base, PRO, and PRO PLUS production versions.
